# FruitWise AI 

FruitWise AI is a professional Android application that uses Artificial Intelligence to detect the freshness of fruits. It can classify whether an **Apple, Mango, or Orange** is **Fresh** or **Rotten** in real-time.

## ✨ Features
* **AI Freshness Detection:** High-accuracy fruit classification using a custom TensorFlow Lite model.
* **Modern UI:** Built with a sleek **Glassmorphism** theme.
* **Smooth Navigation:** Includes a professional Splash screen and Onboarding flow.

## 🛠️ Tech Stack
* **Language:** Java
* **Machine Learning:** TensorFlow Lite (TFLite)
* **Image Processing:** Glide Library
